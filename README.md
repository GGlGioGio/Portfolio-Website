## Portfolio Website
This is a personal portfolio website built with modern web technologies to showcase my projects and educational background.

## Tech Stack
HTML5: Structured content and layout.

CSS3: Responsive design, Flexbox for layout, custom animations, and clean UI components.

JavaScript (Vanilla): Interaction logic and dynamic content handling.

## Project Structure
```
├── 📁 assets          Contains project logos, educational certificates, and media assets.
│   ├── 📁 CV
│   ├── 📁 icons
│   └── 📁 images
├── 📝 README.md
├── 🌐 index.html
├── 📄 script.js       Dynamic functionality and interactivity.
└── 🎨 style.css       All styles, including custom card components and layout configurations.
```
